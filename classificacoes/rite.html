RITE

O que é?

RITE (Rapid Iterative Testing and Evaluation, em português, Avaliação e Teste Iterativos e Rápidos) é uma abordagem que lembra o Teste de Usabilidade. Um pequeno número de usuários (cerca de 5) testa o software ou protótipo e os erros de usabilidade são identificados. A equipe corrige esses erros e o produto é novamente testado. Esse ciclo se repete até obter um software de qualidade.

Materiais Utilizados

Para realizar a abordagem RITE, você vai precisar de:

Software de videoconferência

Quando o RITE for aplicado remotamente. Alguns exemplos: Skype, Zoom, Google Meet, GoTo Meeting, Lookback.
Gravador de Tela

Para capturar as ações do usuário enquanto usa o produto. Alguns sistemas operacionais possuem essa ferramenta.
Gravador de Voz e Vídeo

Quando o RITE for aplicado de forma presencial como câmera, microfone, smartphone.

Como executar a abordagem RITE?

Você deve selecionar os participantes (para cada ciclo, utilize cerca de 5 participantes). Solicite a autorização deles para prosseguir com a pesquisa.

Se o RITE for aplicado de forma presencial, prepare o gravador de tela e gravador de voz e vídeo. Você pode fazer anotações enquanto aplica a abordagem.

Se o RITE for aplicado remotamente, utilize o de software de videoconferência e peça para o usuário compartilhar a tela. Você mesmo pode gravar a tela com o próprio software de videochamada ou de terceiros.

Identifique erros de usabilidade conforme os usuários forem utilizando seu produto. Faça as alterações necessárias com base nos insights colhidos no 1º ciclo de entrevistas.

Com os erros de usabilidade que foram identificados resolvidos, recrute novamente usuários e refaça o teste. A cada ciclo as falhas são percebidas pelos usuários e devem ser solucionadas.
