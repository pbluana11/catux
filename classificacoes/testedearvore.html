Teste de Árvore

O que é?

Teste de Árvore é um método para avaliar a estrutura de um site. Os usuários tentam concluir uma tarefa pré-estabelecida utilizando apenas a estrutura de categorias do site.

Materiais Utilizados

Para realizar o método Teste de Árvore, você vai precisar de:

Gravador de Tela

Para capturar as ações do usuário enquanto usa o produto. Alguns sistemas operacionais e softwares de videoconferência possuem essa ferramenta.
Software de vídeochamada

Para testes aplicados remotamente, alguns exemplos de softwares: Skype, Zoom, Google Meet, GoTo Meeting, Lookback.
Optimal Workshop

A Optimal Workshop possui uma ferramenta chamada Treejack própria para Teste de Árvore.

Como executar o método Teste de Árvore?

Primeiramente, você deve preparar seu site para o teste. Faça uma cópia do seu site e exiba apenas os links de navegação. Remova todos os itens que possam confundir o usuário no momento do teste, deixando-o um site clicável.

Recrute os participantes (entre 5 e 10). Não esqueça de pedir autorização aos usuários e assegurar que seus dados estarão em completo sigilo.

Defina cerca de 10 a 20 tarefas a serem realizadas pelos usuários. Elas tem que ter como objetivo procurar itens usando somente a estrutura do site. O teste tende a ser rápido e pode ser aplicado tanto presencialmente quanto remotamente.

Com os resultados você pode avaliar tanto a estrutura do site quanto a performance do usuário ao realizar as tarefas.
